# 徐码iOS安装方法

## 1. 准备工作

你需要在电脑上的 Rime 中成功部署输入法（目的是预先生成 bin 文件）。或者使用我生成的 bin 文件。

## 2. 安装 iRime

在 App store 中下载 iRime 输入法。

下载完毕后，进入 App，点击「电脑快传」。

在电脑浏览器上输入对应的 ip 地址。

进入文件管理系统。

## 3. 复制文件

将 ```schema_ios/```文件夹中所有文件和文件夹复制到对应的手机文件夹```Rime/```下。

对于已经存在的文件，覆盖之。

## 4. 部署

手机上回到主界面，点击「部署」。成功后即可使用徐码了。
